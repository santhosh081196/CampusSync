# Contributing

Contributions are welcome.

## Local development

1. Fork and clone the repository.
2. Install Node.js 20 LTS or later.
3. Run `npm install`.
4. Run `npm start`.
5. Create a focused branch for your change.
6. Test on the operating system affected by your change.
7. Open a pull request with a clear description and screenshots for UI changes.

## Good first contributions

- Improve accessibility and keyboard navigation.
- Add tests for date grouping and Canvas item normalization.
- Add encrypted credential storage using the operating system keychain.
- Add multi-account support.
- Add native notifications and configurable reminders.

Do not commit OAuth credentials, Canvas access tokens, generated installers, or files from `node_modules`.
